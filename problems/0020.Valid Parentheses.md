Example 1:
Input: s = "()"
Output: true

Example 2:
Input: s = "()[]{}"
Output: true

Example 3:
Input: s = "(]"
Output: false

```python
class Solution:
    def isValid(self, s: str) -> bool:
        dict = {
            ')' : '(',
            ']' : '[',
            '}' : '{'           
        }

        stack = []
        for i in s:
            if i in dict.keys():
                if not stack or stack.pop() != dict[i]:
                    return False
            else:
                stack.append(i)
        return True if not stack else False
```
